# Инструкция по созданию инсталлятора MAX Antivirus

Для создания профессионального установщика (.msi или .exe) мы будем использовать **CPack**, который уже встроен в систему сборки проекта.

### 1. Подготовка (Windows)
Убедитесь, что у вас установлены:
*   **WiX Toolset** (для создания .msi пакетов): [wixtoolset.org](https://wixtoolset.org/releases/)
*   **NSIS** (если вы хотите .exe установщик): [nsis.sourceforge.io](https://nsis.sourceforge.io/)

### 2. Сборка и упаковка
Откройте терминал в папке `build` и выполните следующие команды:

```bash
# 1. Конфигурация проекта с включенным брендингом MAX
cmake .. -DCMAKE_BUILD_TYPE=Release -DCPACK_PACKAGE_NAME="MAX Antivirus" -DCPACK_PACKAGE_VENDOR="MAX Security"

# 2. Компиляция всех компонентов
cmake --build . --config Release

# 3. Генерация инсталлятора
cpack -G WIX
```

### 3. Что будет внутри инсталлятора?
Инсталлятор автоматически соберет:
*   `MAX.exe` — основное приложение.
*   `libclamav.dll` — движок сканирования.
*   `ui_mockup.html` — интерфейс.
*   Необходимые конфигурационные файлы и папки для баз данных.

### 4. Настройка ярлыков
В файл `CMakeLists.txt` добавлены инструкции по установке. После запуска инсталлятора:
*   Программа будет установлена в `C:\Program Files\MAX Antivirus`.
*   Будет создан ярлык на рабочем столе и в меню "Пуск".
